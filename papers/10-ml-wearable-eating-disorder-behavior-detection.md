## Summary

Pilots ML-based detection of eating disorder behaviors in everyday life using passive wearable sensors. Demonstrates that wearable data can identify eating disorder episodes with high accuracy.

## Key Findings

- Machine learning classifiers using passive wearable sensor data accurately detect eating disorder behaviors
- Average classification accuracy of 91% for detecting eating disorder episodes
- Uses physiological signals (HR, HRV, skin temperature, acceleration) from wrist-worn device
- Detection works in everyday life settings, not just laboratory
- Passive sensing requires no user input, enabling continuous unobtrusive monitoring

## MAON Applicability

Directly validates that MAON's wearable signal set can detect eating disorder behaviors. The 91% accuracy using signals MAON has access to (HR, HRV, acceleration) provides a strong foundation for eating disorder monitoring. MAON could identify concerning patterns without requiring explicit user self-report.

Published in Psychological Medicine (2024). Pilot study in everyday life settings. Passive wearable sensing. ML classification approach. High accuracy results.
