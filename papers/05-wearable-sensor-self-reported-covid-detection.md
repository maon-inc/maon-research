## Summary

Combines wearable sensor data (Fitbit) with self-reported symptoms to improve COVID-19 detection accuracy. Demonstrates that combining objective physiological data with subjective reports outperforms either alone.

## Key Findings

- Combining wearable data with self-reported symptoms improves COVID-19 detection over either alone
- Resting heart rate and sleep patterns from Fitbit were key predictive features
- Self-reported symptoms provided complementary information to physiological signals
- The multi-modal approach achieves higher sensitivity and specificity
- Demonstrates value of integrating subjective and objective health data

## MAON Applicability

Validates MAON's core design: combining wearable physiological data with user self-reports via conversation. The finding that multi-modal (objective + subjective) approaches outperform single modalities directly supports MAON's iMessage-based health conversation integrated with Apple Watch data.

## Methodology Notes

Published in Nature Medicine (2021). Uses Fitbit wearable data combined with symptom surveys. Large-scale study. Demonstrates complementary value of subjective + objective data.
