## Summary

Presents the Stress-Predict dataset collected from 35 healthy subjects using wrist-worn PPG sensors during standardized stress tasks (Stroop, TSST, Hyperventilation). Provides a benchmark dataset for wrist-only stress detection similar to MAON's signal availability.

## Key Findings

- Wrist-worn PPG sensors can capture stress-related heart rate changes in 77% of participants
- Respiratory rate changes detected in 80% of participants during stress tasks
- Stress tasks caused significant and measurable physiological variation detectable by consumer-grade wrist sensors
- The Trier Social Stress Test produced the strongest physiological response
- Dataset is publicly available for stress detection model development

## MAON Applicability

Provides a complementary dataset to WESAD for MAON's stress model development. Since MAON uses wrist-only signals (Apple Watch), this wrist-focused dataset is more directly applicable than chest-mounted alternatives.

## Methodology Notes

Pilot study with 35 healthy volunteers published in Sensors (2022). Uses wrist-worn PPG sensor only (no chest device). Standardized stress protocol. Dataset publicly available. Open access.
