## Summary

Develops and validates a real-time alerting system for COVID-19 detection using consumer wearable data. In a cohort of 3,318 participants, the system generated pre-symptomatic alerts for 80% of infected individuals at a median of 3 days before symptom onset.

## Key Findings

- Real-time alerting system detected 80% (67/84) of SARS-CoV-2 infections in a cohort of 3,318
- Pre-symptomatic signals observed median 3 days before symptom onset
- System uses resting heart rate, skin temperature, and activity data
- Anomaly detection approach compares current readings against individual baseline
- Scalable to large populations using consumer wearable data

## MAON Applicability
